refinekit

Leave visual feedback on any web page. Click elements, add notes, and copy them for AI coding tools.

Drag this button to your bookmarks bar:

+ refinekit

Then use it:

  1. Go to any web page
  2. Click the + refinekit bookmark
  3. Click any element to annotate it
  4. Copy your annotations and paste into Claude Code
Can't drag? Create the bookmark manually

Right-click your bookmarks bar, choose "Add bookmark", set the name to refinekit, and paste this as the URL:

javascript:void(function(){var s=document.createElement('script');s.src='https://unpkg.com/refinekit/dist/refinekit.js';document.body.appendChild(s)})()
